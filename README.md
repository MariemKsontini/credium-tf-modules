# Azure Infrastructure Setup - Terraform Modules 

This repository provides a blueprint for Azure infrastructure setup using Terraform. It is designed to serve as a reusable and modular approach, offering Terraform modules for common Azure resources that can be utilized across multiple projects. By organizing these modules in a consistent and reusable manner, this repository simplifies infrastructure management and promotes best practices in IaC (Infrastructure as Code).
- [Azure Kubernetes Service](./aks/readme.md)
- [Azure Key Vault](./key_vault/readme.md)
- [Azure Container Registry](./acr/readme.md)
- [Log Analytics Workspace](./log_analytics/readme.md)
- [Azure resource groups](./rg/readme.md)
